Kurulum

Gereksinimler

  • PHP 5.4 ve üzeri.
  • Apache 2 web sunucusu.
  • PHP GD2 eklentisi.
  • PHP Mysqli eklentisi.
  • Apache Rewrite modülü.
  • Mysql 5 / MariaDB veritabanı.

Manuel Kurulum

Composer ile kurulumu gerçekleştirememeniz durumunda manuel olarak da kurulum yapabilirsiniz.
Kurulumu yapmak için;

  • Newsbull Github üzerinden güncel sürümü indirin.
  • Zip dosyasından tüm dosyaları çıkarın.

Son Adım

Dosya kurulumunu tammalandıktan sonra veritabanı ayarlarını yapmanız gerekmektedir.

Bilgisayarınızda çalışıyorsanız;
application/config/development/database.php dosyasındaki veritabanı bilgilerini güncelleyiniz.

Dosyalarınızı sunucunuza yüklediyseniz;
application/config/production/database.php dosyasındaki veritabanı bilgilerini güncelleyiniz.

Herşey tamamsa tarayıcınızdan Newsbull'u kurduğunuz adresi açarak kurulum sihirbazını çalıştırın. Sayfayı ilk açtığınızda sihirbaz otomatik olarak çalışacaktır. Açılan sayfadaki butona basarak son kalan veritabanı kurulum işlemini de tamamlayarak yazılımı kullanmaya başlayabilirsiniz.

İyide Panel Nerede?

Kurulumu yaptıktan sonra kurulumu yaptığınız adresin soruna /panel ekini ekleyerek panel sayfasına erişebilirsiniz. Yetkili panel kullanıcı bilgileri;
Kullanıcı Adı: root
Parola: root

Bu kullanıcı tüm yetkilere sahiptir ve yetkilendirme sisteminden muaftır. Sistemi kullanacak diğer panel kullanıcıları için özel yetkiler belirleyerek yeni kullanıcılar oluşturabilirsiniz.
